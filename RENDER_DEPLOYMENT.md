## ⚠️ Important Notes:

1. **Free Tier Spin Down**: Render free tier spins down after 15 min of inactivity
   - First request may take 30-60 seconds to wake up
   
2. **MongoDB Atlas**: Make sure to whitelist Render's IPs in MongoDB Atlas
   - Or set "Allow from anywhere" (0.0.0.0/0)

3. **Environment Variables**: Double-check all env vars are set correctly
   - Backend needs MONGO_URI
   - Frontend needs backend URL

4. **Build Time**: 
   - Backend: ~2-3 minutes
   - Frontend: ~5-7 minutes (Next.js builds take time)
   - Admin: ~3-4 minutes

## 🆘 Troubleshooting:

**Backend won't start:**
- Check MONGO_URI is correct
- Verify MongoDB Atlas allows connections

**Frontend shows API errors:**
- Make sure NEXT_PUBLIC_DEV_API_URL points to backend
- Wait for backend to fully deploy first

**Build fails:**
- Check build logs in Render dashboard
- Verify package.json has all dependencies
